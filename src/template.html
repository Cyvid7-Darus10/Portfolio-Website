Student Registration System

It has two modes which are the admin and the student. In the admin you can basically do everything while on student mode the user can only view subjects, enroll in them, drop them and update their record. You can also import a list of students and subjects but when importing a student with an unknown subject enrolled in the program asks the user if he wants to create that subject or import from masterlist or exit where the unknown subject will just be saved as none.
